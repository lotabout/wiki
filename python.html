Python Env setup

PIP is slow downloading use douban's mirror: http://pypi.douban.com/simple/

add the following lines in ".pip/pip.conf" 

[global]
index-url = http://pypi.douban.com/simple/

Install Numpy with mkl support using PIP add mkl options to "~/.numpy-site.cfg" 

[mkl]
library_dirs = /opt/intel/composer_xe_2013.1.117/mkl/lib/intel64
include_dirs = /opt/intel/composer_xe_2013.1.117/mkl/include
mkl_libs = mkl_rt
lapack_libs =
