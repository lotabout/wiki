Quick Sort

  • Quick Sort

Quick sort, also known as qsort, is a sorting algorithm based on partition. In normal case, its complexity is O(n logn), however with worst case, it is O(n^2).

The structure of qsort is: 

void qsort(l, u) 
{
    if (l >= u)
        return
    m = l
    for i = [i+1, u]
        if (x[i] < x[l])
            swap(++m, i)
    swap(l, m)
    qsort(l, m-1)
    qsort(m+1, u)
}
