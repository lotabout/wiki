Windows Server 2008 无法启用网络发现

必须打开以下服务：dnscache（简写、fdrespub（简写）、SSDP Discovery、UPnP Device Host、Computer Browser、Server、DNS Client、TCP/IP NetBIOS Helper。

如果最后网络发现和文件共享两个项都能选为开启，但是仍然无法在网络中显示出其他用 户的图标，那是因为没有开启本机的guest用户帐号，没有赋予权限，局域网其他用户无 法登录进入网内。
