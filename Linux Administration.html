Music Player

In console:

crontab

Q1: python script not running. A1: All crontab commands are run as a string to sh -c, which however will stick PWD to HOME.

Q2: how to find X settings in cron A2: Check Find Session ID in CronTab 

dbus_session_file=~/.dbus/session-bus/$(cat /var/lib/dbus/machine-id)-0
if [ -e "$dbus_session_file" ]; then
  . "$dbus_session_file"
  export DBUS_SESSION_BUS_ADDRESS DBUS_SESSION_BUS_PID
  dbus-send …
fi
