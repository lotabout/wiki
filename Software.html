conky

  • conky shows "0%" for "${mixer}" => in my system, the mixer is actually controled by 'xfce4-mixer'. So, run 'xfce4-mixer' and select the channels you want to control, then re-start conky, and it should be OK now.
  • make conky show disk io usage. => conky has options for disk io: \(diskio \)diskio_read $diskio_write
