Slackware Upgrade

NOTE: These steps are done without the help of slackpkg+

  1. Exclude kernel in "/etc/slackpkg/blacklist"
  2. Exclude 3rd party repositories by adding appropriate lines for their repo tag.
  3. Install new kernel. 
    installpkg a/kernel-*.t?z
  4. update slackpkg's inner database.
  5. upgrade slackpkg, If we don't do this, it will abort after upgrading itself

during the steps. 

slackpkg upgrade slackpkg
slackpkg new-config

Overwrite the old config file.

  1. re-configure "/etc/slackpkg/blacklist"
  2. Let slackpkg do all the upgrades. 
    slackpkg install-new
slackpkg upgrade-all
slackpkg clean-system

Multilib Support

  1. Download Alien Bob's multilib. 
    rsync -av rsync://taper.alienbase.nl/mirrors/people/alien/multilib/14.0/ multilib-14.0/
  1. install/upgrade the existing gcc and glibc packages and compat32-tools. 
    cd multilib-14.0
upgradepkg --install-new *.t?z
  1. Upgrade the set of converted 32-bit Slackware packages(often referred to as the "compat32" packages. 
    upgradepkg --install-new slackware64-compat32/*-compat32/*.t?z
