My own window manager. Borrowed ideas from dwm and i3wm.

Design

Features needed

  • status bar + system tray (dwm+systray patches)
  • minimize window and area to show these windows
  • dynamic layout management
  • title bar(perhaps?)

Concepts

  • Workspace: Organize stasks using virtual desktop
    • workspaces are created/destroyed dynamically (like i3)
  • frame: Place to place windows.
    • mechanism to implement minimize area
    • A workspace normally have main frame and minimize frame.
    • no dynamic creation/destruction/management support for now
  • Layout: dynamic layouts(like dwm)
    • attached to every frame.
  • XCB/XLib ?
    • XCB is the latest preference. Nevertheless lacks good documentation.
