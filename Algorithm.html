Permutation

Q1: given a sequence [1,2,3,4,5] for example, and a number n < len(seq)!, output a new sequence which is the n-th permutation of the original sequence.

For example, given a sequence [1,2,3], and a number n = 4, then the output sequence should be [3,1,2]

because the permutations of [1,2,3] are: 0 -- [1 2 3] 1 -- [1 3 2] 2 -- [2 1 3] 3 -- [2 3 1] 4 -- [3 1 2] -- this is the requested seq. 5 -- [3 2 1]

Q2: given a sequence and a permutation of it, find out the index of the permutation in all its permutations.

=> this is the inverse question of Q1.

given sequence [1,2,3] and its permutation [3,2,1], we should get 4.

For Q1 and Q2, refer to Lehmer Code.
