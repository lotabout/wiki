Sed Cookbook

Some common tasks solved by sed.

Dealing with Lines

  1. Deleting Empty Lines 
    sed '/^$/d' < filename
  1. Merging Empty Lines into One 
    sed '/./,/^$/!d' < filename

Numbering

  1. numbering each line 
    sed = filename | sed 'N; s/\n/\t/'
  1. count line numbers of a file (wc -l) 
    sed -n '$=' < filename

Text Conversion and Substitution

  1. convert dos files(with \r\n) to unix files(with \n) 
    sed 's/.$//' < filename
sed 's/^M//' < filename
sed 's/\x0D$//' < filename
  2. convert unix newline(\n) to dos format(\r\n) 
    sed 's/$/\r' # gsed 3.02.80 or higer
sed 's/$/'"/`echo \\\r`/" # commandline under bash
sed 's/$/\x0D/' # commandline under bash
    the key is to express \r.
